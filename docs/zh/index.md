# SCL-Core

高性能生物算子库

## 简介

SCL-Core 是一个专为生物数据分析设计的高性能算子库，提供零开销抽象、SIMD 加速和默认并行化的计算内核。

## 快速开始

```bash
# 安装
pip install scl-core

# 使用
import scl
```

## 特性

- **零开销抽象** - 编译为最优机器代码
- **SIMD 加速** - 向量化计算内核
- **默认并行** - 自动并行化
- **内存高效** - 稀疏矩阵基础设施
