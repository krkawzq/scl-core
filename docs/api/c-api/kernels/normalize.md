Matrix normalization operations.

## Overview

Functions for normalizing sparse matrices by rows or columns.

## Functions

### `scl_normalize_rows()`
Normalize matrix rows.

### `scl_normalize_cols()`
Normalize matrix columns.

### `scl_compute_row_norms()`
Compute row norms without normalization.

## Norm Types

- `SCL_NORM_L1` - L1 norm
- `SCL_NORM_L2` - L2 (Euclidean) norm
- `SCL_NORM_MAX` - Max norm
